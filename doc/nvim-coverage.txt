================================================================================
INTRODUCTION                                                   *nvim-coverage*

nvim-coverage displays lcov code coverage in the sign column with highlight
groups and a summary popup. It reads a pre-generated lcov file — it does not
run tests or invoke any external tools itself.

  1. Setup................................................|nvim-coverage-setup|
  2. Commands.........................................|nvim-coverage-commands|
  3. Lua API...........................................|nvim-coverage-lua-api|

================================================================================
SETUP                                                    *nvim-coverage-setup*

                                                            *coverage.setup()*
coverage.setup({opts})
    Initial setup function. Must be called before using the plugin.

    Usage: >

    require('coverage').setup({
      -- file = "coverage/lcov.info",  -- override if needed
      auto_reload = { enabled = true },
    })
<

Valid keys for {opts}:

    file: ~
        Path (string) or list of paths (table) to the lcov file. When a list
        is given the first existing file is used. Used by |:CoverageLoad| and
        |coverage.load()| when no file argument is given.
        Defaults to: >
            { "lcov.info", "cover/lcov.info", "coverage/lcov.info", "target/lcov.info" }
<

    auto_reload: ~
        Table controlling file watching behaviour.

        auto_reload.enabled: ~
            When true, the lcov file is watched for changes after loading.
            Signs are updated automatically whenever the file changes on disk.
            The watcher stops when |:CoverageClear| or |coverage.clear()| is
            called.
            Defaults to: `false`

        auto_reload.timeout_ms: ~
            Debounce delay in milliseconds before reloading after a change.
            Defaults to: `500`

    commands: ~
        When true, register the |nvim-coverage-commands| on setup.
        Defaults to: `true`

    on_load: ~
        A Lua function called after coverage is loaded.
        Example: >

        require('coverage').setup({
            on_load = function()
                vim.notify("Coverage loaded")
            end,
        })
<
    sign_group: ~
        Name of the sign group used when placing signs. See |sign-group|.
        Defaults to: `"coverage"`

                                                         *coverage.highlights*
Valid keys for {opts.highlights}:

    Each highlight group accepts: `fg`, `bg`, `sp`, `style`, `link`.
    See |highlight-gui| and |:highlight-link|.

    covered: ~
        Highlight for covered lines.
        Defaults to: `{ fg = "#B7F071" }`
    uncovered: ~
        Highlight for uncovered lines.
        Defaults to: `{ fg = "#F07178" }`
    partial: ~
        Highlight for partially covered lines (branch coverage).
        Defaults to: `{ fg = "#AA71F0" }`
    summary_border: ~
        Border highlight for the summary popup.
        Defaults to: `{ link = "FloatBorder" }`
    summary_normal: ~
        Normal text highlight for the summary popup.
        Defaults to: `{ link = "NormalFloat" }`
    summary_cursor_line: ~
        Cursor line highlight for the summary popup.
        Defaults to: `{ link = "CursorLine" }`
    summary_header: ~
        Header row highlight for the summary popup.
        Defaults to: `{ style = "bold,underline", sp = "fg" }`
    summary_pass: ~
        Highlight for files above the min_coverage threshold.
        Defaults to: `{ link = "CoverageCovered" }`
    summary_fail: ~
        Highlight for files below the min_coverage threshold.
        Defaults to: `{ link = "CoverageUncovered" }`
    virtual_text: ~
        Highlight for virtual text hit counts.
        Defaults to: `{ link = "Comment" }`

                                                              *coverage.signs*
Valid keys for {opts.signs}:

    covered: ~
        Sign for covered lines.
        Defaults to: `{ hl = "CoverageCovered", text = "▎" }`
    uncovered: ~
        Sign for uncovered lines.
        Defaults to: `{ hl = "CoverageUncovered", text = "▎" }`
    partial: ~
        Sign for partially covered lines.
        Defaults to: `{ hl = "CoveragePartial", text = "▎" }`

    Each sign also accepts an optional `priority` key. See |sign-priority|.

                                                            *coverage.summary*
Valid keys for {opts.summary}:

    width: ~
        Width of the popup window as a fraction of the terminal width.
        Defaults to: `0.70`
    height: ~
        Height of the popup window as a fraction of the terminal height.
        Defaults to: `0.50`
    min_coverage: ~
        Coverage percentage threshold. Files below this value are highlighted
        with `summary_fail`, files above with `summary_pass`.
        Defaults to: `80.0`
    borders: ~
        Border characters for the popup. See plenary window API.
    window: ~
        Additional window options passed to the popup. See plenary window API.

                                                        *coverage.virtual_text*
Valid keys for {opts.virtual_text}:

    enabled: ~
        When true, virtual text hit counts are shown automatically after
        |coverage.load()|.
        Defaults to: `false`

    position: ~
        Where to place the virtual text. One of `"eol"`, `"right_align"`,
        or `"inline"`. See |nvim_buf_set_extmark()|.
        Defaults to: `"eol"`


================================================================================
COMMANDS                                              *nvim-coverage-commands*

:CoverageLoad [{file}]                                         *:CoverageLoad*
    Load an lcov file and place signs immediately. If {file} is not given,
    uses the `file` configuration option.

:CoverageShow                                                  *:CoverageShow*
    Show coverage signs. Requires a prior |:CoverageLoad|.

:CoverageHide                                                  *:CoverageHide*
    Hide coverage signs.

:CoverageToggle                                              *:CoverageToggle*
    Toggle coverage sign visibility.

:CoverageClear                                                *:CoverageClear*
    Remove signs, clear the cached report, and stop the file watcher.

:CoverageSummary                                            *:CoverageSummary*
    Open the coverage summary popup. Requires a prior |:CoverageLoad|.

:CoverageToggleHitCounts                            *:CoverageToggleHitCounts*
    Toggle virtual text hit counts on/off. Requires a prior |:CoverageLoad|.


================================================================================
LUA API                                                *nvim-coverage-lua-api*

                                                             *coverage.load()*
coverage.load({file}, {place})
    Load an lcov file. If {file} is nil, uses `config.opts.file`.
    When {place} is true, signs are placed immediately; otherwise they are
    cached and shown when |coverage.show()| is called.
    Also starts the file watcher when `auto_reload.enabled` is true.

                                                            *coverage.show()*
coverage.show()
    Show coverage signs. No-op if coverage has not been loaded or signs are
    already visible.

                                                            *coverage.hide()*
coverage.hide()
    Hide coverage signs.

                                                          *coverage.toggle()*
coverage.toggle()
    Toggle sign visibility.

                                                           *coverage.clear()*
coverage.clear()
    Remove signs, clear the cached report, and stop the file watcher.

                                                         *coverage.summary()*
coverage.summary()
    Open the coverage summary popup.

    Summary popup keybindings:
        `s`       Sort by coverage ascending
        `S`       Sort by coverage descending
        `H`       Jump to top entry
        `<CR>`    Open file under cursor
        `?`       Toggle help screen
        `q`       Close
        `<Esc>`   Close

                                                *coverage.toggle_hit_counts()*
coverage.toggle_hit_counts()
    Toggle virtual text hit counts. Shows execution count for every
    instrumented line as virtual text (e.g. `× 42`). Requires a prior
    |coverage.load()|.

                                                       *coverage.jump_next()*
coverage.jump_next({sign_type})
    Jump to the next sign of {sign_type} in the current buffer.
    {sign_type} is one of `"covered"`, `"uncovered"`, `"partial"`.
    Defaults to `"covered"`.

                                                       *coverage.jump_prev()*
coverage.jump_prev({sign_type})
    Jump to the previous sign of {sign_type} in the current buffer.
    {sign_type} is one of `"covered"`, `"uncovered"`, `"partial"`.
    Defaults to `"covered"`.
